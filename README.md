TWRP device tree for ASUS Zenfone 3 (ZE520KL)
========================================================

For building TWRP for MSM8953 models only.

Manifest: 
https://github.com/minimal-manifest-twrp/platform_manifest_twrp_omni/tree/twrp-8.0

To build:
```sh
. build/envsetup.sh
lunch omni_Z017-userdebug
mka recoveryimage
```
